- **Frameworks**

- **Libraries**

- **Languages**

## Awesome, but how can I use it?

You can invite the bot with [this link](https://discord.com/oauth2/authorize?client_id=937476981321240626&permissions=139586825280&scope=bot%20applications.commands) or click the invite button on [discode.mazy.tk](https://discode.mazy.tk).

This bot makes use of slash commands. Just type `/`, locate the bot icon, enjoy (hopefully).
